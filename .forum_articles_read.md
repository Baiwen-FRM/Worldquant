## 工作流模板：论坛阅读 → 丰富 Skills

> 每次开新窗口执行此任务时，按以下流程操作：

### 1. 前置检查
- 读取本文件 `.forum_articles_read.md`，确认已读文章列表
- 跳过所有标记为 `[x]` 的文章

### 2. 搜索新文章
```python
# 使用 cnhkmcp 搜索中文论坛
cnhkmcp.search_all_forum_slow(
    search_query="<关键词>",
    locale="zh-cn",
    max_results=20
)
```
**推荐搜索关键词组合**（每轮换一组）：
- `Alpha 优化 Sharpe 提交`
- `IS ladder Sharpe 优化`
- `turnover 降低 优化`
- `Python alpha 提交 经验`
- `Sub-universe Sharpe`
- `Robust Sharpe 优化`
- `operator 使用 技巧`
- `store 模式 数据字段`

### 3. 去重
- 将搜索结果中的 article_id 与本文件 `已读文章` 列表比对
- 跳过已读文章

### 4. 阅读新文章
- 用 `read_full_forum_post(article_id)` 逐篇阅读
- 优先阅读高票数（vote_sum 高）的文章

### 5. 标记已读
- 阅读后立即更新本文件，添加一行 `- [x] <id> - <title>`
- 如果文章非常有价值，在末尾标注票数，如 `(79票, 必读)`
